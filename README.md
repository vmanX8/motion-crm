# MotionCRM

A modern client management dashboard built with React, TypeScript, and GSAP.

A small but realistic frontend simulation of a CRM/admin panel. The goal was to focus on clean UI, smooth interactions, and a structure that could scale.

---

## Techology Stack

- React (with TypeScript)
- Vite
- Tailwind CSS
- GSAP (animations)
- jQuery (used in an isolated legacy-style component)

---

## Features

- Responsive dashboard layout (sidebar + topbar)
- Animated UI using GSAP (page load, cards, transitions)
- Clients table with basic structure
- Clean and reusable component architecture
- Routing with React Router
- Dark modern UI

---

## Purpose

This project is part of my frontend portfolio and focuses on:

- building structured React applications
- combining modern tools (React) with animation libraries (GSAP)
- demonstrating the ability to integrate legacy patterns (jQuery) when needed
- creating UI that feels like a real product instead of a simple demo

---

## Notes

The app uses mock data and does not connect to a backend.  
The focus is on frontend architecture, interaction, and visual quality.
